** Task Management System | Java

---> A console-based Task Manager built using core Java OOP principles and Data Structures.

** Features
-> Add, remove, and update tasks
-> Priority-based scheduling using Min-Heap (PriorityQueue)
-> Undo last action using a Linked List Stack
-> Filter tasks by status or priority
-> O(1) task lookup via HashMap

** Data Structures Used
| Structure | Purpose |
 
| HashMap | Fast task lookup by ID |
| Stack (LinkedList) | Undo functionality |
| PriorityQueue (Min-Heap) | Process highest priority task first |

 ** How to Run
```bash
javac *.java
java Main
```

 ** Tech Stack
Java | OOP | Collections Framework
